# Save The World

## 1. 개요

* 장르: 포인트 앤 클릭
* 플랫폼: Windows
* 개발 환경 / 엔진: Unity
* 빌드 버전: v1.0.0

---

## 2. 실행 방법

### Windows

1. 압축 파일 해제
2. `SaveTheWorld.exe` 실행

---

## 3. 조작 방법

* 마우스와 키보드 사용

---

## 4. 저장 데이터

* 저장 위치: C:\Users\<사용자이름>\AppData\LocalLow\SaveTheWorld\SaveTheWorld\

* 자동 저장 여부: "옵션" -> "메인메뉴"로 이동할 때, 그리고 "엔딩" 장면으로 갈 때 자동 저장

* 초기화 방법: "메인메뉴"에서 "새 게임 시작" 하기, 또는 세이브파일 삭제
